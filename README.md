# Медицинский дневник

Веб-приложение для отслеживания приема лекарств с напоминаниями и интеграцией с календарем.

## Функции

- Отслеживание приема лекарств по дням
- Уведомления о времени приема
- Адаптивный дизайн для мобильных и десктоп
- Локальное хранение данных
- Регистрация и авторизация
